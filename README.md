# EcoRAG PubMed

Energy-aware retrieval-augmented generation experiments for PubMed-style biomedical literature workflows.

## Structure

- `app/` - application entrypoints
- `data/` - raw, processed, and cached datasets
- `notebooks/` - exploratory analysis
- `src/` - reusable project code
- `outputs/` - generated run artifacts
- `reports/` - benchmark summaries and writeups

## Setup

```powershell
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```
